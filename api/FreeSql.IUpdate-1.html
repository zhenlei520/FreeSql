    BatchOptions(Int32, Int32, Boolean)

    批量执行选项设置，一般不需要使用该方法

    各数据库 rows, parameters 限制不一样，默认设置：

    MySql 500 3000

    PostgreSQL 500 3000

    SqlServer 500 2100

    Oracle 200 999

    Sqlite 200 999
